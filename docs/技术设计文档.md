# 5. 核心SQL设计

## 5.1 商品详情聚合

```sql
SELECT
    MIN(price) AS minPrice,
    MAX(price) AS maxPrice,
    (SELECT price
     FROM price_records
     WHERE goods_id = :goodsId
     ORDER BY record_date DESC
     LIMIT 1
    ) AS latestPrice
FROM price_records
WHERE goods_id = :goodsId;
```

---

## 5.2 价格趋势

```sql
SELECT record_date, price
FROM price_records
WHERE goods_id = :goodsId
ORDER BY record_date ASC;
```

---

## 5.3 商店比价

```sql
SELECT s.name, pr.price, pr.record_date
FROM price_records pr
JOIN store s ON pr.store_id = s.id
WHERE pr.goods_id = :goodsId
ORDER BY pr.price ASC;
```

---

## 5.4 商店商品聚合

```sql
SELECT
    g.id,
    g.name,
    MIN(pr.price) AS minPrice,
    MAX(pr.price) AS maxPrice,
    MAX(pr.record_date) AS lastBuyDate
FROM price_records pr
JOIN goods g ON pr.goods_id = g.id
WHERE pr.store_id = :storeId
GROUP BY g.id;
```

---

## 5.5 仪表盘统计

```sql
SELECT COUNT(*) FROM goods;
SELECT COUNT(*) FROM store;
SELECT SUM(price) FROM price_records;
```

# 17. 最终工程约束

* Room为唯一数据源
* Repository为唯一数据入口
* ViewModel禁止直连DAO
* OCR必须输出RawText→Parser链路
* Compose必须单向数据流
* 所有模块可独立测试
